Configure FireStick
  • Step 1: Enable 3rd-Party Apps

First, you need to enable the FireStick to install third-party apps. This section will walk you through all the steps.

Power on the device and go to the `Settings` tab at the top of the screen.

Next, scroll over to the `Device` icon and select it, then select `Developer Options`.

Scroll down to `Apps from Unknown Sources` and turn it `ON`, then confirm.

Quick-Nav:

Power ON > Settings > Device > Developer Options > Apps from Unknown Sources > Turn `ON`
`Downloader` App
  • Step 2: Install `Downloader` from the Amazon App Store

On top of the FireStick home screen, go to the `Search` tab and search for `Downloader`. After `Downloader` installs, open it. The screen will look like this:

Now you are ready to install Titanium.

Quick-Nav:

Home > Search > input `Downloader` > Press `GO`
Download `TitaniumTV` APK
  • Step 3: Use `Downloader` to download the Titanium APK file

Click in the `url` field, then enter:


https://www.firesticktricks.com/titv

then click the `Go` button.

Installation should take about a minute. When the app is finished downloading, click `DONE`

Make sure to Delete the APK file after completing installation.

Quick-Nav:

Open `Downloader` > Search `firesticktricks.com/titv` > Scroll down and click `Install`
Choose a Media Player
  • Step 4: Install an External Media Player

Now, you need to install a media player to play the videos on your FireStick. The Recommended App is MX Player. Open your `Downloader` app again and search:


http://bit.ly/mxfire

like in Step 3. Follow the instructions to install MX Player.

Quick-Nav:

Open `Downloader` > Search `http://bit.ly/mxfire` > Scroll down and click `Install`
Configure Titanium
  • Step 5: Run Titanium for the First Time

When you run the Titanium TV app on FireStick for the first time, you are greeted with the Changelog window. When you are past it, you see a Disclaimer window. Here, you need to click Accept. Next, you are asked to choose the default video player. Select MX Player to make it the default player.

You can always change this in Settings.

Quick-Nav:

Open Titanium TV > click `Next` > Scroll down to `MX Player` > Click to set as `Default Video Player`
